# Vellum Design Library

A shared component library for the [Vellum](https://github.com/vellum-ai/vellum-assistant) ecosystem, built with React 19, Tailwind CSS v4, and Radix UI primitives.

## Getting Started

Install the package in your app:

```bash
bun add @vellum/design-library
```

Import components directly:

```tsx
import { Button, Card, Typography } from "@vellum/design-library";
```

Or use subpath imports for targeted access:

```tsx
import { Button } from "@vellum/design-library/components/button";
```

## Design Tokens

The library ships CSS custom properties via `tokens.css`. Import it in your app's CSS entry point:

```css
@import "@vellum/design-library/tokens.css";
```

Tokens adapt to light, dark, and velvet themes via the `data-theme` attribute on a parent element.

## Theming

Set the active theme by adding `data-theme` to your root element:

```html
<html data-theme="dark">
```

Available themes: `light` (default), `dark`, `velvet`.

Use the toolbar above to preview components in each theme.

## Contributing

- Components live in `packages/design-library/src/components/`
- Each component is a single file with co-located stories
- Run `bun run storybook` to develop locally
- Run `bun run typecheck` to verify types

See [`AGENTS.md`](https://github.com/vellum-ai/vellum-assistant/blob/main/packages/design-library/AGENTS.md) for component conventions and review checklist.
